# Microblog

这是一个用于练习flask的博客项目，参考教程为Miguel Grinberg的技术博客：[The Flask Mega-Tutorial](https://blog.miguelgrinberg.com/post/the-flask-mega-tutorial-part-i-hello-world)

### microblog的主要功能如下：
- 注册、登入与登出
- 编辑个人主页、发布博文
- 在广场浏览他人发布的博文、浏览他人主页
- 关注、取关
- 在home页显示关注者动态

### 备注：
- 支持显示最近登陆时间、每条博文发布时间
- 支持显示关注者与被关注者人数
- 支持邮箱重置密码
